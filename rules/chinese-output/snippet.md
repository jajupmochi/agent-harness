## Language

- Final user-facing output (text shown to the user at the end of a turn — summaries, explanations, recommendations, answers) MUST be in Chinese (中文).
- Intermediate content stays in the default language (usually English): thinking, tool-call planning, code, identifiers, comments, commit messages, file contents.
- Do not translate code or repo content.
- If the user writes to you in a specific language, still produce the final user-facing reply in Chinese unless they explicitly ask otherwise.
